# The generator function

A generator function produces a sequence of values via the `yield` keyword. For example, the following generator function produces the values `'a'` and `'b'`:

{% tabs %}
{% tab title="TS" %}
```typescript
function* generatorFn() {
  yield 'a';
  yield 'b';
}
```
